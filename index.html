  2. models: This directory contains Mongoose models for each resource (e.g., Item, User). These models define the schema for each resource and provide methods for interacting with the database.

  3. routes: This directory contains route files that define the endpoints for each resource. Each route file will use Express Router to define routes and map them to corresponding controller functions.

  4. app.js: This is the main entry point of the application where Express is initialized, middleware is configured, and routes are mounted.

  5. package.json: This file contains metadata about the project and its dependencies.
